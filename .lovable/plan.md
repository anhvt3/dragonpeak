Tạo file hướng dẫn sử dụng Lovable dạng Markdown để team có thể tham khảo và chia sẻ.

- Giới hạn (không hỗ trợ Angular, Vue, Next.js, backend trực tiếp)

- Ví dụ prompt tốt vs xấu

**3. Cấu trúc Project khuyến nghị**

**5. Debug & Troubleshooting**
